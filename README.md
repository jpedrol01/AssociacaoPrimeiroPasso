# Associação Desportiva Primeiro Passo — Site de Demonstração (Single Page)

Este projeto foi consolidado em uma única página (`index.html`) com seções para História, Trajetória, Missão e Contato.

Como usar

1. Abra `index.html` no navegador.
2. Edite as seções diretamente em `index.html`.
3. Estilos estão em `css/style.css` e scripts em `js/main.js`.

Observações

- O formulário de contato faz validação front-end e simula o envio (não há backend).
- Para mostrar fotos no site, coloque imagens reais em `assets/images` com os nomes `activity1.jpg`, `activity2.jpg` e `activity3.jpg`.

Se quiser, eu posso configurar deploy no GitHub Pages ou transformar o site em um PWA.
